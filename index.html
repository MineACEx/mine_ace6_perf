OnePlus Ace 6 Performance Tuning Module | Smooth UI · Enhanced Touch · 180Hz Unlock

✨ Core Features

🔓 180Hz Display Unlock: Removes the official 165Hz limit to enable 180Hz high refresh rate. Automatically locks to 165Hz if skipped.
🎯 Global 330Hz Touch Sampling: Boosts touch sampling rate to 330Hz in all scenarios, reducing latency and improving responsiveness.
📦 Background Process Enhancement: Raises the system cached process limit to 64 and locks 25 apps in memory.
🚀 Rendering & Smoothness Optimization: Forces GPU composition and hardware acceleration, maximizes scrolling cache, and disables redundant log I/O.
⚙️ Secure Installation Script: Includes MD5 checksum verification, automatic DTBO backup, and volume-key-controlled flash mode selection.

📋 Installation Guide

1. Ensure your OnePlus Ace 6 has an unlocked bootloader and Magisk installed.
2. Download the module zip file and install it via Magisk.
3. During installation, choose flash mode:
Volume Up = Flash DTBO to unlock 180Hz / Volume Down = Skip and keep 165Hz
4. Reboot your device. The module will take effect automatically.
5. Optional: Install `sxl.apk` in the root directory to customize the screen refresh rate.
